# kneadedwithlove

Homemade sourdough & baked goods — **Kneaded with Love** landing site.

## Run locally

```bash
cd kneaded-with-love
export PATH="$PWD/.node/bin:$PATH"   # if using project-local Node
npm install
npm run dev
```

Open **http://localhost:5173**
